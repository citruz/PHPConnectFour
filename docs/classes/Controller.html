Controller

Der Controller ist dafür zuständig, alle Anfragen vom User auszuwerten, zu validieren und ggf. an das Model weiterzuleiten.

Response

Überprüft den Login-Status des Users.

checkLogin(bool $needsLogin, bool $needsNoLogin)

Arguments

$needsLogin

bool

Wenn der Paramter true ist und der User nicht eingeloggt seien sollte, wird er auf die Login-Seite weitergeleitet.

$needsNoLogin

bool

Wenn Paramter true ist und der User eingeloggt seien sollte, wird er auf die Index-Seite weitergeleitet.
